平院OJ粘贴助手 (Pingyuan OJ Paste Helper)
本项目由AI协作生成
本脚本专为解决平院 OJ 提交页面禁用粘贴的问题而设计，通过物理注入技术直接将代码写入底层节点。

🛠️ 安装步骤
安装环境：

确保你的浏览器已安装 Tampermonkey (油猴) 插件。

安装脚本：

在本仓库中点击进入 平院OJ粘贴助手-18.0.user.js。

点击右上角的 Raw 按钮。

在跳出的油猴界面点击 “安装”。

⌨️ 使用方法
进入页面：打开平院 OJ 的代码提交页面。

快捷键唤醒：按下 Alt + V。

注入代码：在弹出的输入框中粘贴你的 C/C++ 代码并确定。

正常提交：代码注入后，手动输入验证码并点击 OJ 原有的提交按钮即可。
